# Equation Lab

这个目录专门存放“等式实验室”互动小游戏的代码和文档。

## 目录结构

- `src/equation_lab.html`
  - 当前的单文件 HTML 原型
- `docs/README.md`
  - 项目说明与后续维护入口

## 当前原型能力

- 双线结构
- 闯关线：按教学节奏推进预设关卡
- 实验线：支持自定义输入、样例导入、随机出题
- 拖拽项进行等式变形
- 跨等号自动变为相反项
- 拖入倍数括号时按倍数缩放
- 自动合并同类项
- 通用工具箱支持两边同时加减乘除
- 支持撤销、重置、下一关
- 包含整数、括号与简单分数关卡
- 提供动态教练提示

## 运行方式

直接在浏览器中打开 `src/equation_lab.html` 即可。

## 产品决策

- 这不是单纯的题库，而是“双线教学工具”
- 闯关线负责循序渐进，把等式直觉拆成一关一关的小目标
- 实验线负责自由试错，让学生自己输入方程、验证路线、比较不同变形
- 后续应用题能力，建议作为第三层“方程工坊”加入，而不是直接塞进现有自由输入框
