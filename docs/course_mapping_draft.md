## 1. 这份草案怎么读

### 1.1 置信度说明
- `高`：数学目标和课程位置高度一致，可直接挂载。
- `中`：可挂载，但更像桥接关或过渡关，后续可能需要重排。
- `低`：目前只能暂挂，不建议直接拿它定义阶段能力。

### 1.2 三种映射类型
- `稳定映射`：可以作为该单元的正常关卡。
- `桥接映射`：更适合作为阶段尾声、过渡关或补充关。
- `待定 / 占位`：目前没有合适现成关卡，先显式留空。

## 3. 先说不能硬映射的地方

### 3.1 当前主线缺一个真正的“等号初见”模块
根据成熟资料，小学代数启蒙应先把等号教成“关系”，再进入未知数变形。

但当前 `1-1` 到 `1-3` 已经进入了“把常数请走”的操作。

所以：
- `A0 真正的等号初见` 当前没有对应关卡。
- 这里应保留占位符，不要硬说 `1-1` 就是等号初见。

### 3.2 当前“选择未知量”脚手架仍然偏弱
现有 build 关很多，但多数还是在“给卡片、拼式子”。

真正的：
- 先决定谁是未知量
- 再决定如何设

目前没有独立成段。

所以：
- `C3 选择未知量` 保留占位。

### 3.3 线段图桥接目前没有现成关卡
现有关卡还没有“先看关系图，再写等式”的正式设计。

所以：
- `D4 线段图桥接` 保留占位。

## 6. 当前映射后的整体判断

### 6.1 可以直接成立的部分
- `A2`、`B1/B3`、`C1/C2`、`D1/D2`
- `T1`、`T2`、`T3`、`T4`

这些部分基本都能用当前关卡直接支撑。

### 6.2 需要桥接处理的部分
- `A1`：现在已有题太快进入“请走常数”
- `A-Bridge`：可以成立，但要明确它是“列式初见”，不是独立大阶段
- `B-Bridge`：乘法与分数列式可成立，但更像过渡单元

### 6.3 当前最明显的课程缺口
1. 真正的等号关系起点
2. 明确选择未知量的脚手架
3. 方程工坊入门关
4. 主线总复习
5. 线段图桥接

这些都不应假装已经存在。
