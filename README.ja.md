# Equation Lab（等式実験室）

[English](./README.md) | [中文](./README.zh.md)

Equation Lab は、小学校高学年向けのインタラクティブな数学学習サイトです。項をドラッグしたり、まとめたり、展開したり、両辺を同時に変形したりしながら、等式変形と代数の感覚を育てます。

公開ページ：
[GitHub Pages](https://silencecat.github.io/equation-lab/)

デモについて：
[![Equation Lab demo](./docs/assets/equation_lab_demo.gif)](./docs/assets/equation_lab_demo.mp4)

このデモでは、等号の向こうへ項を移す操作、カードから等式を組み立てる操作、道具を等号にドラッグして両辺を同時に変形する操作、括弧の展開をまとめて見せています。

## プロジェクト概要

学習モードは 2 本立てです。

- `クエストモード`：1 つずつ考え方を学ぶ段階式レベル
- `プレイグラウンド`：式を自由に入力して試せる実験モード

単なる問題集ではなく、子どもが自分の手で等式を変形しながら学べる道具を目指しています。

## 主な機能

- カードをドラッグして等式を変形
- 符号の変化を「等価変形の結果」として見せる
- `+`、`-`、`×`、`÷` の道具を両辺に同時に適用
- 整数、簡単な分数、括弧、文章題に対応
- 中国語、日本語、英語の 3 言語対応
- 静的サイトとしてそのまま配布可能
- オフライン用の単一 HTML ファイルを書き出し可能

## 保護者向け

使い方はとても簡単です。

- オンライン版をそのまま開く
- またはオフライン用単一ファイル [`dist/equation_lab_standalone.html`](./dist/equation_lab_standalone.html) を開く

子どもは次のように使えます。

- レベル形式で順に学ぶ
- プレイグラウンドで自分の式を試す
- ひとつの解き方を暗記するのではなく、複数の道筋を比べる

## 開発者向け

このプロジェクトは静的な HTML/CSS/JS アプリで、簡単なビルドで自己完結型の単一 HTML を生成します。

主なディレクトリ：

- [`src/`](./src/) ソースコード
- [`dist/`](./dist/) ビルド成果物
- [`tests/`](./tests/) 単体テストとブラウザ回帰テスト
- [`docs/`](./docs/) 内部ドキュメントとカリキュラム草案

## ローカル開発

```bash
npm install
npm test
npm run test:e2e
npm run build
```

よく使う出力：

- [`dist/index.html`](./dist/index.html)：GitHub Pages 用
- [`dist/equation_lab.html`](./dist/equation_lab.html)：ビルド済み単一ファイル
- [`dist/equation_lab_standalone.html`](./dist/equation_lab_standalone.html)：配布しやすいオフライン用単一ファイル

## 参加方法

次のような協力を歓迎します。

- カリキュラムの難易度調整
- 文章題のアイデア

## 目標

このプロジェクトの目的は、「移項のやり方」を暗記させることではありません。両辺に同じことをすれば、等式はずっと成り立つという感覚を育てることです。
